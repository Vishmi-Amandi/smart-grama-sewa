# Smart Grama Sewa

A digital platform to streamline interactions between citizens and Grama Niladhari officers, enabling online appointments, form downloads, and real-time availability tracking.

##  Project Overview

Smart Grama Sewa aims to reduce waiting times and improve service delivery in Grama Niladhari divisions by providing:
- Online appointment booking
- Real-time officer availability
- Digital form downloads in 3 languages
- AI chatbot for citizen guidance
- Automated notifications
